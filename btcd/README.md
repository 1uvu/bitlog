# Bootstrap a btcd peer

- clone this project: `git clone https://github.com/1uvu/bitlog.git && cd bitlog/btcd`
- modify .env.example with the help of comments: `nano docker-compose/.env.example`
- modify `peers/peer-example/btcd.conf` with the help of comments: `nano docker-compose/peers/peer-example/btcd.conf`
- run bootstrap.sh: `bash bootstrap.sh`
